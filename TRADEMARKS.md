# CutDirector brand policy

`CutDirector` is the project name and brand identifier used by Fangx-AI. This
file does not claim that the name is registered in every jurisdiction.

The software and content licenses in this repository do not grant permission to
use the CutDirector name, logo, account identity, or visual identity in a way
that is likely to imply sponsorship, endorsement, affiliation, or official
status.

## Allowed without separate permission

- Truthfully state that software or content is based on, compatible with, or
  derived from CutDirector.
- Link to the canonical repository.
- Preserve required attribution and historical references.
- Name an unmodified GitHub fork in the manner GitHub normally displays it.

## Permission required

- Use `CutDirector` as the primary name of another product, service, Skill,
  commercial offering, domain, social account, course, or repository that is
  not the canonical project.
- Use the CutDirector visual identity in a confusingly similar logo or store
  listing.
- Claim that a modified version is official, endorsed, certified, or maintained
  by Fangx-AI.

Modified versions should use a clearly different name and state:

> Based on CutDirector by Fangx-AI; this version is independent and is not
> endorsed by the CutDirector project.

`ChatCut` and all other third-party names, logos, and marks belong to their
respective owners. Their appearance in this repository does not imply
endorsement unless the relevant owner expressly states otherwise.

For brand-permission questions, open an issue in the canonical repository:
https://github.com/Fangx-AI/cut-director/issues
